# MineralsCorp Portfolio Website\n\nA professional company portfolio website for a minerals processing company built with Next.js, TypeScript, and Tailwind CSS.\n\n## Features\n\n- **Modern Design**: Professional, clean design inspired by leading mining industry websites\n- **Responsive Layout**: Fully responsive design that works on all devices\n- **Smooth Scrolling**: Single-page application with smooth scrolling navigation\n- **Professional Sections**:\n  - Hero section with company statistics\n  - About section with company history and achievements\n  - Services section showcasing mineral processing capabilities\n  - Products section featuring mineral offerings\n  - Contact section with form and company information\n- **Admin Login**: Secure login page for portfolio management\n- **TypeScript**: Full TypeScript support for better development experience\n- **Tailwind CSS**: Utility-first CSS framework for rapid styling\n\n## Tech Stack\n\n- **Frontend**: Next.js 14, React 18, TypeScript\n- **Styling**: Tailwind CSS\n- **Icons**: Lucide React\n- **Animations**: Framer Motion (ready to implement)\n- **Future Backend**: PostgreSQL + TypeORM (planned)\n- **File Storage**: Cloudinary CDN (planned)\n\n## Getting Started\n\n### Prerequisites\n\n- Node.js 18+ installed\n- npm or yarn package manager\n\n### Installation\n\n1. Navigate to the project directory:\n   ```bash\n   cd company-profile-app\n   ```\n\n2. Install dependencies:\n   ```bash\n   npm install\n   ```\n\n3. Run the development server:\n   ```bash\n   npm run dev\n   ```\n\n4. Open [http://localhost:3000](http://localhost:3000) in your browser\n\n### Available Scripts\n\n- `npm run dev` - Start development server\n- `npm run build` - Build for production\n- `npm run start` - Start production server\n- `npm run lint` - Run ESLint\n\n## Project Structure\n\n```\nsrc/\n├── app/\n│   ├── globals.css          # Global styles\n│   ├── layout.tsx           # Root layout\n│   ├── page.tsx            # Home page\n│   └── login/\n│       └── page.tsx        # Admin login page\n├── components/\n│   ├── Navbar.tsx          # Navigation component\n│   ├── HeroSection.tsx     # Hero section\n│   ├── AboutSection.tsx    # About section\n│   ├── ServicesSection.tsx # Services section\n│   ├── ProductsSection.tsx # Products section\n│   ├── ContactSection.tsx  # Contact section\n│   └── Footer.tsx          # Footer component\npublic/\n└── images/                 # Static images\n```\n\n## Design Features\n\n### Color Scheme\n- **Primary**: Professional blue tones (#0284c7 - #0c4a6e)\n- **Secondary**: Neutral grays (#1c1917 - #fafaf9)\n- **Accent**: Warm oranges (#ea580c - #fff7ed)\n\n### Typography\n- **Font**: Inter (Google Fonts)\n- **Headings**: Bold, large sizes with proper hierarchy\n- **Body**: Clean, readable text with good contrast\n\n### Components\n- **Navigation**: Fixed header with dropdown menus\n- **Hero**: Full-screen with statistics and call-to-action\n- **Sections**: Well-spaced with consistent padding\n- **Cards**: Hover effects and smooth transitions\n- **Forms**: Professional styling with validation states\n\n## Customization\n\n### Adding Images\n1. Place images in `public/images/` directory\n2. Update components to reference the images\n3. Consider using Next.js Image component for optimization\n\n### Updating Content\n- Edit component files in `src/components/`\n- Update company information, services, and products\n- Modify color scheme in `tailwind.config.js`\n\n### Adding New Sections\n1. Create new component in `src/components/`\n2. Import and add to `src/app/page.tsx`\n3. Update navigation in `Navbar.tsx`\n\n## Future Enhancements\n\n### Phase 2: Backend Integration\n- [ ] PostgreSQL database setup\n- [ ] TypeORM configuration\n- [ ] User authentication system\n- [ ] Admin dashboard for content management\n\n### Phase 3: File Management\n- [ ] Cloudinary integration\n- [ ] Image/document upload functionality\n- [ ] File management system\n- [ ] Support for multiple file formats\n\n### Phase 4: Advanced Features\n- [ ] Blog/news section\n- [ ] Project gallery\n- [ ] Client testimonials\n- [ ] Multi-language support\n- [ ] SEO optimization\n- [ ] Analytics integration\n\n## Contributing\n\n1.
